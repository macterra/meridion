The following smilies have come from a variety of sources. Some have been downloaded from sites across the Internet, some are heavily modified versions of readily available smilies, others have been created from scratch. All of them have been altered to make a "set" of smilies, each with the same size, shape, colour and style.

Feel free to use or alter these smilies as you wish. I only ask that you upload these smilies to your own server, and do not link directly to the smilies on this site. A zip file containing all the smilies and this index page can be downloaded HERE.

If you would like to contact me, my email address is jason@bhra.org.uk.
