Example of use
 Bold [b] [/b] e.g.
[b]Some Text[/b]
Some Text
Italic [i] [/i] e.g.
[i]Some Text[/i]
Some Text
Underline [u] [/u] e.g.
[u]Some Text[/u]
Some Text
Strikethrough [s] [/s] e.g.
[s]Some Text[/s]
Some Text
Glow [glow=color,glow width, #characters wide] [/glow] e.g.
[glow=red,2,300]Some Text[/glow] Some Text
Shadow [shadow=color,glow width, #characters wide] [/shadow] e.g. [shadow=red,left,300]Some Text[/shadow]Some Text
Scroll (Move) [move] [/move] e.g.
[move]Some Text[/move]
Some Text
Preformatted Text [pre] [/pre] e.g.
[pre]     Some Text[/pre]
    Some Text
Left aligned text [left] [/left] e.g.
[left]Some Text[/left]

Some Text
Centered text [center] [/center]

e.g.
[center]Some Text[/center]

Some Text
Right aligned text [right] [/right]

e.g.
[right]Some Text[/right]

Some Text
Horizontal line [hr]   e.g. [hr]
Text Size [size=2] [/size] Some Text
Font [font=Verdana] [/font] Some Text
Flash Image [flash=200,200] [/flash] URL of flash image
Web Address [url]
or [url=http://somewhere]		 [/url] e.g.
[url=http://site.com]Some Text[/url]
[url]http://site.com[/url]
FTP Address [ftp] [/ftp] e.g.
[url=ftp://site.com]Some Text[/url]
[url]ftp://site.com[/url]
Image Address [img] [/img] URL of gif, jpg or png
Mail Address [email] [/email] mailto:someone@somewhere
Table [table]
[tr]
[td]		 [/td]
[/tr]
[/table]		 Some Table Stuff
Insert Colum [tr] [/tr] Some Table Stuff
Insert Row [td] [/td] Some Table Stuff
Superscript [sup] [/sup] e.g.
Some[sup]Superscript[/sup] Text
SomeSuperscript Text
Subscript [sub] [/sub] e.g.
Some[sub]Superscript[/sub] Text
SomeSuperscript Text
Teletype [tt] [/tt] e.g.
[tt]Some fixed spacing text[/tt]
Some fixed spacing text
Code (i.e. don't perform formatting) [code] [/code] e.g.
[code]Some [b] Text[/b][/code]
Some [b] Text[/b]
Quotation [quote] [/list] e.g.
[quote]Some Text[/quote]
Some Text
Indented List [list] [/list] e.g.
[list]Some Text[/list]
        Some Text
Set Color [color=ColorName] [/color] e.g.
[color=Green]Some Text[/color]
Some Text
